# apparel-lookbook-intro

テンプレ **`apparel_lookbook`（テンプレ15）** 用の固定 HTML。

- **オープニング**: 固体スカイブルー＋**下寄せ**のロゴスタック（上に大余白）→ 角丸カード（**ロゴは上にはみ出し可**／`.alb-intro-card-clip` で写真のみクリップ）→ 全画面 **レターボックス**＋テキストは画像**下寄せ** → 動画。
- **キャンペーン青ブロック**（`04-full-page-scroll` 参照）: `GLOBAL WORK × NAOKO TAKAYAMA`・シーズン英字・和文リード・上丸の縦写真。
- **ルック**: 白背景・大余白、**細いセリフ番号**＋和文サブ（`.alb-look-ja`）、スライダー右下に **「1 / 3」＋番号ボタン**（半透明白ボックス）、**薄グレー帯**＋ **Check**（ピル型）。
- **参照画像**: `docs/references/apparel-lookbook/*.png`
- **10セクション**（参照ロングの英語見出し）＋中間のフル幅動画ブロック＋フッター **LOOK** 大タイポ。
- **コードは独自実装**です。本番では文言・画像・価格・リンクを差し替えてください。

## 参照スクショを静的デザインに寄せるには

チャットに貼った画像だけでは、別スレの AI が参照できません。**`docs/references/apparel-lookbook/`** に PNG/JPG を置き、同梱の `README.md` に色・レイアウトメモを書いてから「照合して CSS を直して」と依頼すると、ピクセル寄せがしやすくなります。
