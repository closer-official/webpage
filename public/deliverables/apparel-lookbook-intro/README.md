# apparel-lookbook-intro

テンプレ **`apparel_lookbook`（テンプレ15）** 用の固定 HTML。

- **オープニング**: 固体スカイブルー＋中央スタック → 角丸カードが下から出現し **写真の上に白文字（EiiS スタック）** → 全画面化時 **上下にスカイのレターボックス** → 動画。
- **ルック**: 白背景・大きな縦余白、見出し **`01 | ON THE MORNING ROAD` 形式**（セリフの大きい番号）、**全幅**メイン画像、右下 **1 2 3**、**薄グレー帯**の中に **Check** ボタン（白地・細枠）。
- **10セクション**（参照ロングの英語見出し）＋中間のフル幅動画ブロック＋フッター **LOOK** 大タイポ。
- **コードは独自実装**です。本番では文言・画像・価格・リンクを差し替えてください。
