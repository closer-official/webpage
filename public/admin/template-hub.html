公開の流れ（ビルトイン）

  • templateRegistry.js に ID を追加し、buildHtml 等を実装（§4a）。
  • 新規 ID を 下書きに: galleryDraftBuiltins（Supabase app_store またはローカル JSON）の draftBuiltinIds に追加 → 一般ギャラリー非表示。またはデプロイ後このページのカードで ギャラリーを非表示。
  • 下の 「ギャラリーに公開」で非表示解除 → 一般向けカタログ・ヒアリングに載ります。
