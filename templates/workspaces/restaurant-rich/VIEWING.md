# このテンプレの「いま」をどこで見るか

| 段階 | 見る場所 |
|------|----------|
| 作業中（ID 未登録） | このフォルダの `mock/index.html` をブラウザで開く（ローカルファイル） |
| ID がカタログに載ったあと | **`/admin/template-hub.html`**（一覧＋iframe）· **`/api/template-preview/<id>`**（全画面） |
| お客様と同じ一覧 | **`/template-gallery`** |

ギャラリーに **ビルトインとして載せる**手順は `docs/テンプレ制作ワークスペース要件書.md` の **§4a**。  
ワンクリックではなく、**ハブページの「文面をコピー」→ Cursor に貼る**と毎回同じやり方にできます。
