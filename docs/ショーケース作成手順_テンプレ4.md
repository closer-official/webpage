# テンプレ2（カフェ・喫茶・パン・スイーツ）ショーケースの作り方

**全9種類のテンプレのうち、テンプレ2（旧 Warm Organic・カフェ／パン統合）**の素材を整え、完成データをリポジトリに残します。

データの正本は **`src/data/warm-organic-showcase.json`** です（`warmOrganicCafePreset.ts` がこれを読み込みます）。

---

## 手順 A: JSON を直接編集（推奨）

1. `src/data/warm-organic-showcase.json` をエディタで開く。
2. `content`（店名・見出し・セクション・`heroSlides`・`footerAddress` 等）と `seo` を書き換える。
3. 保存後、**デザイン確認**タブの Warm Organic プレビューで見た目を確認。

---

## 手順 B: 手動調査でテンプレ4のLPを作る

1. **手動・詳細**タブで Maps からカフェをキューに追加し、「調査」を開く。
2. テンプレートで **カフェ・喫茶・パン・スイーツ（2）** を選び、コンセプト等を入力してダッシュボードへ。
3. 必要ならエクスポートや HTML を元に JSON を手作業で `warm-organic-showcase.json` 形式に合わせて反映。

---

## 参照データの渡し方（1種類ずつ貼る）

**文章と写真は 1 種類ずつ**、チャットやアプリに貼って渡す運用にします。

- **文章** … 参照サイトの URL・業種・「いいと思ったポイント」をそのままテキストで送る。
- **写真** … 参照したい画像を 3～5 枚、その種類の分だけ貼る。

種類ごとに「この種類の文章＋写真」を 1 回ずつ送ってもらえれば、こちらでショーケース用のデータに反映します。まとめて PDF やフォルダで用意する必要はありません。

---

## 補足

- メインのデモ用実データは **フルオート**タブで地域・カテゴリを指定して作成する想定です（カテゴリごとに1件など）。
- ヒーロー3枚は JSON の `content.heroSlides` で指定します。
