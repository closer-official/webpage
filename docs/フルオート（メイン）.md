# フルオート（メイン）

## 何をするか

1. **地域**（例: 東京都港区、茨城県つくば市）
2. **カテゴリ**（例: カフェ、ホテル、ジム）— プリセットまたは自由入力
3. **作成件数**（1〜25）と **最低レビュー数**

を指定し、**開始**を押すだけです。

サーバーが次を**連続実行**します（手操作不要）。

- Google で「地域 + カテゴリ」を検索し、**公式サイト未掲載**の店だけ取得
- 口コミを Gemini で分析（コンセプト・強みなど）
- **LP をテンプレ3種**で HTML 生成、**DM（メール）文面**も生成
- すべて **ダッシュボード**に追加

あとは **ダッシュボード**タブでプレビュー確認・文面修正・送信前チェックのみ行います。

## 必要な環境

- **server/.env**: `GOOGLE_MAPS_API_KEY`, `GEMINI_API_KEY`
- **フロント .env**: `VITE_API_URL`（開発時は `http://localhost:3001`）
- サーバー起動（`npm run server` 等）

## デモ用データ

カテゴリごとに **件数 1** で回すと、テンプレ傾向の違いが分かりやすいサンプルが揃います。

## API

- `POST /api/full-auto/start` … `{ region, category, count, minReviews }`
- `GET /api/full-auto/status` … 進捗（ポーリング用）

※ ジョブ状態はサーバー**プロセス内メモリ**です。サーバーを複数台にすると共有されません（デモ・単一サーバー想定）。
