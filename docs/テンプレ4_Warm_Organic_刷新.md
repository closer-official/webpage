# テンプレ4（Warm Organic）— 設計の理由（更新）

各パーツは **「なぜそうするか」** を説明できるものだけ採用する方針（チープに見える装飾は削除）。

## 本文・セクション

| 要素 | 理由 |
|------|------|
| **左の縦ライン（廃止）** | 意味のないアクセントはノイズ。セクション区切りは **極細の下線のみ**。 |
| **先頭ブロック（lede）** | ページの「顔」なので見出しを **1.3rem級**、本文 **1rem・行間広め**。地色はページと同じ（**帯で囲まない**）。 |
| **小見出し（営業時間・アクセス等）** | **0.8125rem・字間**でラベル化。本文より一段階下げて階層をはっきり。 |
| **営業時間（`hours`）** | みずたま型の「読みやすさ」は **時間の1行目だけ** に下線マーカー風ハイライト（`.wo-text-mark`）。**枠全体を色塗りしない**（蛍光ペン＝行単位）。2行目以降は通常文。 |
| **引用** | 左右の太線や塗りは重い。**上下の細線＋中央寄せイタリック**だけに抑える。 |

## SNS

| 要素 | 理由 |
|------|------|
| **絵文字（📸 / 💬）** | 「IG」は非専門家に伝わらない。絵文字は **意味が一瞬で伝わり**、ブランドSVGを無断で使わずに済む。 |
| **横並び** | 縦積みはタップ領域が長く無駄に見える。**flex row・適度な間隔**。 |
| **「タップで開きます」** | アクセシビリティと期待値の明示。 |

## フォーム

| 要素 | 理由 |
|------|------|
| **ラベル上・入力フル幅** | インラインラベルは狭い画面でがたつく。**ブロック＋十分な padding（0.9rem）・1.06rem 字**. |
| **textarea 最小高さ 11rem** | 短文でも圧迫感を減らし、長文を書きやすく。 |
| **フォーカスリング** | アクセント色の薄い outline で **どこにいるか** を明示。 |

## ヒーロー・ナビ・フッター（従来どおり）

- **カルーセル＋ドット＋スワイプ**: 複数枚の世界観提示と再確認。
- **固定FAB**: スクロールしても迷子にしない。
- **ヒーロー左ストライプ・フッター同色**: 写真トーンがバラついても **ブランド色で一本化**（意味のある繰り返し）。
- **フッター緑**: ナビFAB・ストライプと同じ `--tp-brand` で **終端のまとまり**。
