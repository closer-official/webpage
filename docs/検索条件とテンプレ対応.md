# 検索条件とテンプレートの自動対応

Maps で検索した**キーワード**（例: `港区 ホテル`）と、店舗の**業種 category**（Google の type）を合わせて、調査モーダルで**初期テンプレ**が決まります。LP 作成時は**そのテンプレの完成例レイアウト**をそのまま使い、店名・コンセプト・強み・アクセスだけ差し替えます。

## 対応表（先に当たったルールを採用）

| テンプレ | キーワード例 |
|---------|-------------|
| **6 High Energy** | ジム、gym、フィットネス、パーソナル、トレーニング、ピラティス、ヨガスタジオ など |
| **5 Pop Friendly** | キッズ、保育、あそび場、ボールプール、playground、preschool など |
| **1 Minimal Luxury** | ホテル、旅館、lodging、エステ、スパ、サロン、resort など |
| **2 Dark Edge** | バー、bar、ラウンジ、ナイトクラブ、居酒屋、ワインバー など |
| **4 Warm Organic** | カフェ、cafe、コーヒー、喫茶、ベーカリー、レストラン など |
| **3 Corporate Trust** | クリニック、歯科、コンサル、税理、法律、不動産、保険 など |

どれにも当たらない場合は **3（Corporate Trust）** を初期値にします。

手動でキューに入れた店は `searchQuery` がないため、**業種（category）欄**の文字だけで推定します（例: 業種に「ジム」と書けば 6）。

調査画面では推奨テンプレを表示しつつ、プルダウンで**別テンプレに変更**することもできます。
