# アパレル・ルックブック参照（EiiS 等）

## なぜ「チャットのスクショだけ」だと静的デザインが寄らないか

- **会話に貼った画像**は、別スレ・別日の AI からは**見えません**（プロジェクトのファイルになっていないため）。
- 実装は `public/deliverables/apparel-lookbook-intro/index.html` の **推定レイアウト**であり、参照元の**ピクセル・トークンと照合した状態ではありません**。

**結論：** 参照を **このフォルダに画像ファイルとして置く**と、以降の修正で「並べて合わせる」作業が可能になります。

---

## やること（最短）

1. **このフォルダに PNG / JPG を保存する**（ファイル名は英数字推奨、例 `01-opening-blue.png`）。
2. 各画像に対応する **1行メモ** を下の表に追記する（「何の画面か」だけで可）。
3. **DevTools またはスポイト**で拾えるものをメモする（なくても可だが精度が上がる）:
   - スカイブルー・帯グレー・フッター LOOK の **#色**
   - 見出し（`01 | TITLE`）の **おおよその px / 相対サイズ感**
   - カード角丸・写真アスペクトの **印象**（細め / 太め など）

4. Cursor で「`docs/references/apparel-lookbook/` の画像と `apparel-lookbook-intro` を照合して CSS を直して」と依頼する。

## チェックリスト（コピーして表の下に貼る）

```
【色】
- 背景スカイ: #
- ショップ帯: #
- フッター LOOK グレー: #

【タイポ】
- 番号「01」: セリフ系 / サンセリフ / サイズ感
- 英語小見出し: トラッキング（広い・狭い）
- スクリプトロゴ: 太さ・傾きの印象

【レイアウト】
- 左右余白: 狭い / 標準 / 広い
- スライド番号: 右下のみ / その他
- 商品サムネ: 正方形に近い / 縦長 など
```

## 著作権の注意

参照画像は**レイアウト・比率・雰囲気の合わせ込み用**です。公開用 LP では、**自社・許諾済みの写真・ロゴ**に差し替えてください。
