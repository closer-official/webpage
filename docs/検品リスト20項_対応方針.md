# 究極検品リスト20項 — 対応方針（LPテンプレート）

静的HTML＋テンプレートCSSで実装。Next.js は本プロジェクト未使用（Vite＋サーバー生成HTML）。

| 領域 | 項目 | 実装 |
|------|------|------|
| 視覚 | タイポ階層（ジャンプ率） | `--type-display` / h2 / リード / 本文の clamp 比を拡大 |
| 視覚 | 字間 | テンプレごとに nav / h2 / subheadline の letter-spacing を最適化 |
| 視覚 | 余白30%+ | section-rhythm を拡大・main の上下余白を増加 |
| 視覚 | 配色（純白黒回避） | 各テンプレでオフホワイト・チャコール系に統一 |
| 視覚 | 画像演出 | Ken Burns 継続＋ヒーロー背景の scroll 連動パララックス |
| 質感 | 多層シャドウ | `--shadow-lift` 等の CSS 変数で soft multi-layer |
| 質感 | 0.5px ボーダー | `border-width: 0.5px` + `@supports not` で 1px フォールバック |
| 質感 | アイコン整合 | ナビハンバーガー線幅をテンプレの線のトーンに合わせる（CSS） |
| 質感 | ノイズ | `body::after` に SVG ノイズ（テンプレで blend 調整） |
| 質感 | ボタン押下 | `:active` で translateY + 影の沈み込み |
| 動的 | イージング | `cubic-bezier(0.22,1,0.36,1)` を scroll-in / hover に統一 |
| 動的 | ホバー | nav / CTA に letter-spacing や transform の微加算 |
| 動的 | スクロール連動 | `data-scroll-in` 全テンプレ＋ヒーロー parallax JS |
| 動的 | ロード | `prefers-reduced-motion` 以外で body フェードイン |
| 動的 | マイクロ | dark_edge ナビ開閉は既存 transition 強化 |
| 信頼 | 認知負荷 | skip-link・見出し階層・#contact 導線維持 |
| 信頼 | レスポンシブ | 既存の overflow-x / 768px ブレーク継続 |
| 信頼 | 爆速 | 画像 lazy / 最小JS（LP は静的1ファイル） |
| 信頼 | a11y | `prefers-reduced-motion` でアニメ停止、:focus-visible 強化 |
| 信頼 | CTA | `.cta-block` / フッター contact を視覚的に強調 |
